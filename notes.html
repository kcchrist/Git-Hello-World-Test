In order to push a new local repo to my GitHub page
Make the new repo: $ git init
Add the Files in the Current Directory to that repo: $ git add .
Commit the repo with a new comment descriptor tag: $ git commit -m "First Commit"
Set up online repo and get the link from the QuickSetup Page
Connect these repos and publish to the public page with: $ git remote add origin *URL HERE*
$ git remote -v
Push these changes online: $ git push -u origin master
FROM: https://help.github.com/articles/adding-an-existing-project-to-github-using-the-command-line/
